## Call in `Main() normal mode`

```C++
int main() {
    game voltorb;
    voltorb.play();
    return 0;
}
```

## Call in `Main() machine mode`

```C++
int main() {
    game voltorb;
    ai_player william;
    voltorb.play(william);
    return 0;
}
```
